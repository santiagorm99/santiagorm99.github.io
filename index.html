Proyectos destacados (topic: showcase)

Record Video GG: aplicacion de mensajería

Aplicación de proyecto de curso creado para el curso de DAM, donde en el cual tiene los siguientes parametros: - Creación de usuario e inicio de sesión: En este apartado una vez iniciada la aplicación puede iniciar sesión si ya hay un usuario previo registrado y si no es el caso se le rederigirá a otra pantalla para crear un usuario con contraseña para que así la base de datos la almacene. - Selección de idiomas: Tiene un botón en el cuál el usuario puede cambiar el idioma al que esté disponible y automaticamente se le cambiará al idioma seleccionado. - Selección de envío: El objetivo es que el usuario seleccione el tipo de envío que desee realizar ya sea carta o paquete y una vez realizado se le creará un id unico en el cual puede insertar en la siguiente pantalla para verificar si el pedido se ha tramitado, esta en camino o ha llegado a su destino.
